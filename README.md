# AGE CHECKER

Hey folks, I'm sharing a project that you can place on your website to notify users that the content it contains is intended for adults, with self-declaration and storage via browser cookies. 
It's actually quite simple, but what matters is that it works.

## 🚀 Features

Function 1 - It asks the user accessing the page a simple self-declaration question with 2 answers: Yes and No. 
If the answer is “No,” the function redirects the user to google.com, and if the user tries to access the website again, it blocks access via browser cookies and displays a blocking notification on the screen. 
The user is blocked via COOKIES for 365 days.


Function 2 - If the user confirms they are of legal age, the function redirects them to a folder in your project and allows access for 30 (thirty) days without displaying the screen again (provided the user is using a regular browser tab, as the decision is saved via cookies).

## 📦 Setting-up

1 - Place it in your project folder and modify it as needed.

2 - Configure your host so that this requirement is randomized regardless of the URL accessed, for enhanced browsing security.
